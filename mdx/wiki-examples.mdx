## Заголовки(после они отображаются справа)

Используй `#`, `##`, `###`, чтобы делить статью на понятные части. 

# Заголовок первого уровня
## Заголовок второго уровня
### Заголовок третьего уровня

## Обычный текст

Пиши обычный текст без специальных символов. Для выделения используй:

- `**жирный текст**` → **жирный текст**
- `*курсив*` → *курсив*
- `` `команда` `` → `команда`

## Списки

Маркированный список:

- Первый пункт
- Второй пункт
- Третий пункт

Нумерованный список:

1. Первый шаг
2. Второй шаг
3. Третий шаг

## Видео

Видео из `wiki/public`:

```mdx
<video src="example-video.mp4" title="Название видео" />
```

Видео по прямой ссылке:

```mdx
<video src="https://samplelib.com/mp4/sample-5s.mp4" title="Видео-гайд" />
```

## Встраивание iframe

Например, YouTube, карта или другой embed:

```mdx
<iframe
  src="https://youtu.be/LVkTWfWanmk"
  title="YouTube video"
/>
```

## Плашки

```mdx

  Напиши здесь важное предупреждение.
</Important>

<Additional>
  Напиши дополнительную информацию.
</Additional>
```
