## Заголовки

Используйте `#`, `##`, `###`, чтобы делить статью на понятные части.

# Заголовок первого уровня
## Заголовок второго уровня
### Заголовок третьего уровня

## Обычный текст

Пишите обычный текст без специальных символов. Для выделения используйте:

- `**жирный текст**` → **жирный текст**
- `*курсив*` → *курсив*
- `` `команда` `` → `команда`

## Списки

Маркированный список:

- Первый пункт
- Второй пункт
- Третий пункт

Нумерованный список:

1. Первый шаг
2. Второй шаг
3. Третий шаг

## Видео

Видео из `wiki/public`:

```mdx
<video src="example-video.mp4" title="Название видео" />
```

Видео по прямой ссылке:

```mdx
<video src="https://samplelib.com/mp4/sample-5s.mp4" title="Видео-гайд" />
```

## Встраивание iframe

Например, YouTube, карта или другой embed:

```mdx
<iframe
  src="https://youtu.be/LVkTWfWanmk"
  title="YouTube video"
/>
```

## Таблицы

| Команда | Что делает |
| --- | --- |
| `/spawn` | Возвращает на спавн |
| `/msg ник текст` | Отправляет личное сообщение |
| `/rules` | Открывает правила |

## Плашки

```mdx
<Important>
  Напишите здесь важное предупреждение.
</Important>

<Additional>
  Напишите дополнительную информацию.
</Additional>
```

<Important>
  Напишите здесь важное предупреждение.
</Important>

<Additional>
  Напишите дополнительную информацию.
</Additional>
