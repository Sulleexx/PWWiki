# Сборка модов

Здесь вы можете скачать нашу сборку модов и узнать, как её установить.

## Скачать сборку

[Скачать сборку Fabric 26.1.2 (26.1.2)](https://drive.google.com/file/d/19fDLZzqWvc6w63MjTO02zBt7KplBp1ph/view)

## Про сборку

Наша сборка работает на загрузчике **Fabric** и версии игры **26.1.2**.

Она включает в себя:
- **Технически важные моды** — Fabric API, Fabric Language Kotlin, Cloth Config, Mod Menu.
- **Необходимые моды** — EmoteCraft (эмоции), PlasmaVoice (голосовой чат), CustomPlayerModels (3D скины).
- **Моды на оптимизацию** — Sodium, Entity Culling, Ferrite Core, Lithium, MemoryLeakFix, LazyDFU, Starlight, Krypton.
- **Рекомендуемые моды** — Iris + Indium (шейдеры), LambDynamicLights, Xaero's Minimap (мини-карта).

## Инструкция по установке

1. Установите **Fabric** для версии 26.1.2.  
   Большинство лаунчеров позволяют сделать это в два клика. В ином случае скачайте установщик с официального сайта Fabric.

2. Скачайте архив сборки по ссылке выше.

3. Откройте архив — внутри будут файлы `.jar` (моды).

4. Перейдите по пути:  
   `%AppData%\Roaming\.minecraft\mods`  
   (если папки `mods` нет — создайте её).

5. Перенесите **все** файлы из архива в папку `mods`.

6. Запустите игру. Если в меню появилась вкладка **«Моды»** и в ней больше 10 модов — установка прошла успешно.

7. Сборка установлена. Приятной игры!
